# 🌟 Python Greeter App

A lightweight command-line application that greets the user and displays the current date and time. This project serves as a perfect boilerplate for simple Python CLI tools.

## ✨ Features
* Prompts the user for their name.
* Delivers a personalized greeting.
* Fetches and displays the current system time.
